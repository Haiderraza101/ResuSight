EXECUTIVE SUMMARY
================================================================================
ResuSight is a comprehensive AI-powered resume classification system that 
leverages an ensemble of 7 different machine learning and deep learning models 
to automatically categorize resumes into relevant job categories. The system 
combines traditional ML approaches (Logistic Regression, SVM, Random Forest), 
state-of-the-art deep learning architectures (BiLSTM variants with attention 
mechanisms), and transformer-based models (DistilBERT) to provide robust, 
diverse, and highly accurate predictions with confidence scores.

The project demonstrates complete full-stack development including a modern 
Next.js React frontend, Flask REST API backend, and integrated ML/DL inference 
pipeline. All 7 models are fully integrated and operational, with professional 
UI displaying predictions from all models simultaneously in an organized, 
color-coded interface.

DATA PROCESSING PIPELINE
================================================================================

INPUT PROCESSING FLOW:
1. FILE UPLOAD
   └─ User uploads resume (PDF or TXT format)
   └─ File validation: type checking, size limit (25MB)

2. TEXT EXTRACTION
   └─ PDF: PyPDF2 library extracts text
   └─ TXT: Direct text reading
   └─ Result: Raw resume text

3. TEXT CLEANING & PREPROCESSING
   └─ Remove URLs and email addresses
   └─ Strip special characters
   └─ Normalize whitespace
   └─ Convert to lowercase
   └─ Result: Clean, standardized text

4. FEATURE VECTORIZATION
   ├─ ML Models: TF-IDF vectorization
   ├─ DL Models: Convert to word sequence using word2idx
   └─ Transformer: Tokenize using DistilBertTokenizer

5. PARALLEL MODEL INFERENCE
   ├─ Logistic Regression prediction
   ├─ Linear SVM prediction
   ├─ Random Forest prediction
   ├─ BiLSTM+Attention prediction
   ├─ DL CNN models (if available)
   └─ DistilBERT prediction

6. RESULT AGGREGATION
   └─ Collect predictions from all 7 models
   └─ Calculate confidence scores
   └─ Generate top-5 predictions per model

7. RESPONSE FORMATTING
   └─ JSON response with all model predictions
   └─ Include confidence scores and probabilities
   └─ Return extracted text for verification

OUTPUT FORMAT:
{
  "success": true,
  "filename": "resume.pdf",

  "extractedText": "...",
  "predictions": {
    "Logistic Regression": {
      "prediction": 3,

      "confidence": 0.87,
      "top5": [
        {"category": "Software Development", "probability": 0.87},
        {"category": "Data Science", "probability": 0.08},
        ...
      ]
    },
    ...
    [6 more models...]
  }
}

BACKEND IMPLEMENTATION DETAILS
================================================================================

FLASK API ENDPOINTS:
───────────────────────────────────────────────────────────────────────────

1. POST /api/resume/upload
   Purpose: Upload resume file and get predictions
   
   Request:
   - Method: POST
   - Content-Type: multipart/form-data
   - Body: file (PDF or TXT)
   
   Response (Success):
   - Status: 200 OK
   - Body: JSON with all 7 model predictions
   
   Response (Error):
   - Status: 400/500
   - Body: JSON with error message

2. POST /api/resume/predict
   Purpose: Get predictions from raw text
   
   Request:
   - Method: POST
   - Content-Type: application/json
   - Body: {"text": "resume text..."}
   
   Response: Same as /upload endpoint

3. GET /health
   Purpose: Check API health status
   
   Response:
   - Status: 200 OK
   - Body: {"status": "healthy", "message": "Flask API is running"}

MODEL LOADER ARCHITECTURE:
───────────────────────────────────────────────────────────────────────────
Location: resusight-backend/model_loader.py
Size: 380+ lines

Key Methods:
• __init__(): Initialize device (CPU/CUDA), load all models
• load_models(): Load ML, DL, and Transformer models with error handling
• clean_text(): Text preprocessing and cleaning
• text_to_seq(): Convert text to token sequences for DL models
• predict_all(): Main inference method returning all 7 predictions
• get_top_n_predictions(): Extract top-5 predictions with probabilities

Error Handling:
• Missing models logged but don't crash system
• Architecture mismatches gracefully skipped
• Console output shows model status indicators
• Fallback mechanisms for optional models


INSTALLATION & DEPLOYMENT
================================================================================

PREREQUISITES:
• Python 3.10+ (conda environment resusight2)
• Node.js 18+ with npm
• Git for version control
• 2GB+ disk space for models and dependencies

BACKEND SETUP:
1. Navigate to resusight-backend folder
2. Ensure conda environment activated:
   conda activate resusight2
3. Install dependencies:
   pip install -r requirements.txt
4. Start Flask server:
   python app.py
5. Server runs on http://localhost:5001

FRONTEND SETUP:
1. Navigate to resusight-frontend folder
2. Install Node dependencies:
   npm install
3. Create .env.local with:
   NEXT_PUBLIC_API_URL=http://localhost:5001
4. Start development server:
   npm run dev
5. Application runs on http://localhost:3000

ENVIRONMENT CONFIGURATION:
File: resusight-frontend/.env.local
